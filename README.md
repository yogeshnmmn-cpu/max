# Mini CRM

A lightweight, single-page CRM built with plain HTML/CSS/JS.

## Features

- Add contacts with name, email, company, sales status, and follow-up date.
- Search across name/email/company.
- Advance status through sales stages with one click.
- Delete contacts.
- Persist data in `localStorage`.

## Run locally

```bash
python3 -m http.server 8000
```

Open <http://localhost:8000>.
